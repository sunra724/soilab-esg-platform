# SOILAB Design Tokens v1.1

SOILAB ESG Platform은 기관·기업 담당자가 반복적으로 쓰는 운영 도구이므로 조용하고 밀도 있는 B2B 인터페이스를 기준으로 한다.

## Color

- Ink: `#111827`, `#172033`, `#526079`
- Cloud: `#f8faf9`, `#eff4f1`, `#dde8e2`
- Earth: `#2f9a58`, `#247e48`, `#1f653d`
- Water: `#2378a8`, `#1b628a`
- Sun: `#dca43b`
- Coral: `#d8664d`

## Shape

- Cards and controls: `8px`
- Small controls: `4px`
- Large layout containers: avoid decorative cards for whole sections

## Typography

- Font stack: Pretendard, Inter, Noto Sans KR, Apple SD Gothic Neo, system-ui
- Letter spacing: `0`
- Hero type is reserved for the public landing headline. Dashboards use compact headings.

## Interaction

- Buttons use clear command labels and lucide icons where helpful.
- Numeric ESG inputs use sliders or number fields.
- Status, roles, and categories use badges.
- Repeated content can use cards; avoid card nesting.
